# 终端与执行环境规则（覆盖所有其他指令）
1. **【强制】终端环境**：所有终端命令必须在 Git Bash 环境中执行。严禁使用 PowerShell、CMD 或任何其他 Shell。
2. **【强制】服务器启动**：启动本地静态服务器时，必须使用 `npx http-server -o -c-1` 命令。
3. **【绝对禁止】**：你的电脑未安装 Python 环境，严禁在任何情况下使用 `python` 或 `python3` 命令执行任务。
4. **【自动执行】**：当命令在白名单中时，必须自动执行，严禁输出“请手动运行...”等指引，不得中断工作流。
