# AI文章生成工作台 - 需求共识文档

## 1. 需求描述

### 1.1 产品定位
AI文章生成工作台是一个帮助用户快速创建高质量文章的工具，通过AI辅助完成文章大纲生成和正文撰写。

### 1.2 核心功能清单

| 功能模块 | 功能描述 | 优先级 |
|---------|---------|-------|
| 主题输入 | 用户输入文章主题 | 高 |
| 大纲生成 | AI基于主题生成文章大纲 | 高 |
| 大纲编辑 | 用户可编辑、确认大纲 | 高 |
| 正文生成 | AI逐段流式生成正文内容 | 高 |
| 生成控制 | 支持暂停/继续/重新生成 | 高 |
| 内容编辑 | 润色/扩写/缩写当前段落 | 中 |
| Markdown导出 | 导出带YAML front matter的Markdown文件 | 高 |
| 项目管理 | 多项目管理，localStorage存储 | 高 |
| 项目搜索 | 支持项目搜索功能 | 中 |

### 1.3 禁止功能
- 不需要登录注册
- 不需要代码审查、代码补全功能
- 不需要聊天助手、文档检索功能
- 不需要历史版本管理
- 不需要分类标签功能

## 2. 技术方案

### 2.1 技术栈
- **前端**: Vue 3 + Vite + TailwindCSS 3
- **后端**: Node.js + Express
- **AI服务**: 兼容OpenAI格式，对接.env中的智谱/Zhipu API
- **数据存储**: localStorage（前端）

### 2.2 API配置
- 使用.env中已配置的ZHIPU_API_KEY
- 后端代理请求，避免前端直接暴露API密钥

## 3. 验收标准

### 3.1 功能验收
1. 用户输入主题后，点击生成按钮能获取AI生成的文章大纲
2. 用户可以编辑大纲，调整标题和段落结构
3. 点击生成正文后，AI逐段流式输出内容
4. 支持暂停、继续、重新生成操作
5. 支持对当前段落进行润色、扩写、缩写
6. 点击导出按钮可下载Markdown文件（带YAML front matter）
7. 多个项目可保存到localStorage，支持搜索筛选

### 3.2 性能验收
1. 大纲生成响应时间 < 5秒
2. 正文流式输出流畅，无明显卡顿
3. 项目列表加载 < 1秒

## 4. 任务边界
- 仅实现上述核心功能
- 不涉及用户认证、权限管理
- 不涉及服务端持久化存储（仅localStorage）
